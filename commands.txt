# Instructions to run the MERN Web Project

# You will need two separate terminals to run this project.

# --- Terminal 1: For the Backend ---

# 1. Navigate to the backend folder from the project root
cd backend

# 2. Install dependencies for the backend
npm install

# 3. Start the backend server
npm start


# --- Terminal 2: For the Frontend ---

# 1. Navigate to the frontend folder from the project root
cd frontend

# 2. Install dependencies for the frontend
npm install

# 3. Start the frontend development server
npm start

# After running the commands, the frontend should be accessible at http://localhost:3000 
# and the backend server will be running on the port specified in its configuration (e.g., http://localhost:5000).
